# AND STARS — Target Clients

Who we sell to, and the hook that lands for each. Use these to write industry-specific landing sections, portfolio categories, and cold outreach.

| Industry | Pain we solve | Angle / hook |
|----------|---------------|--------------|
| **Restaurants** | No online menu, no online orders | Menu + reservations + online ordering; look premium, fill tables |
| **Hotels** | Outdated site, no direct bookings | Booking engine, gallery, reduce OTA commission |
| **Dentists** | Weak local presence | Appointment booking, trust/reviews, local SEO |
| **Lawyers** | Looks unprofessional online | Authority design, case results, consultation booking |
| **Car rental companies** | Manual bookings | Fleet catalog + online booking + payment (custom web app) |
| **Schools** | Hard to manage info/admissions | Info site → school management system upsell |
| **Gyms** | No membership/class flow | Class schedules, memberships, sign-ups |
| **Startups** | Need to launch fast, look funded | Landing pages, MVP web apps, fast iteration |

## Messaging rules
- Lead with **their** outcome (more bookings / sales / patients), not our tech.
- Each industry gets its own landing section or page on the site.
- Portfolio should be grouped by industry — even placeholder/demo builds count early on.

## Priority for first clients
Start with the industries easiest to reach locally and quickest to close: **restaurants, dentists, gyms** (small sites, fast turnaround, clear ROI). Use wins there to fund bigger custom-app work (car rental, schools, hotels).
